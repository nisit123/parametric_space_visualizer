# Parametric Space Visualizer

Interactive MATLAB GUI tool to visualize the relationship between 
Real Space and Parametric Space for 5 geometric forms.

## Features
- Dual-panel plot: Real Space (left) vs Parametric Space (right)
- 5 presets: Circle, Ellipse, Helix, Cylinder, Sphere
- Arc length computation via trapezoidal numerical integration
- Surface area computation via cross-product double integral
- Timer-based animation with synchronized highlight markers
- Live equation display for both spaces

## Tech Stack
MATLAB R2019b or later
